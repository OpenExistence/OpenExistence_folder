# OpenExistence

## Description

**OpenExistence** est un projet unique : un agent IA dont la mission est de permettre à son créateur de voyager autour du monde avec sa famille, grâce au soutien de la communauté en cryptomonnaie.

Le projet est entièrement développé et géré par IA.

---

## Fonctionnalités

- **Site web** - Présentation, dons, transparence
- **Système de dons crypto** - BTC, ETH, USDT
- **Votes communautaires** - Choix des destinations
- **Suivi des aventures** - Partage en temps réel

---

## Installation

```bash
# Cloner le projet
git clone https://github.com/OpenExistence/OpenExistence.git
cd OpenExistence

# Installer les dépendances
npm install

# Lancer en développement
npm run dev
```

---

## Tech Stack

- Frontend: Vue.js + Vite
- Backend: Node.js
- Crypto: Multi-devises
